# 수리딩어학원 - Sureading Academy

## 영어 도서관 시스템 백엔드

### 주요 기능
- 🎯 레벨테스트 (1-5단계, 각 60문제)
- 📚 도서 학습 관리 (챕터별 테스트)
- 📰 영자신문클래스 (실시간 기사 + 이해도 테스트)
- 📊 학습 진도 추적 및 분석
- 📧 월간 리포트 자동 발송

### 수업 유형
- **채두윜반**: 기초 영어 독서
- **소설클래스**: 영어 소설 심화 학습
- **내신클래스**: 학교 내신 대비
- **영자신문클래스**: 실전 영어 + 시사상식

### 기술 스택
- **Backend**: Flask, SQLAlchemy
- **Database**: SQLite
- **Authentication**: JWT
- **Email**: Flask-Mail + Gmail SMTP
- **Deployment**: PythonAnywhere
